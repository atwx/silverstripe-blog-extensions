# SilverStripe Blog Extensions

WORK IN PROGRESS

Extension for the [SilverStripe Blog module](https://github.com/silverstripe/silverstripe-blog) with additional features.

## Requirements

* SilverStripe ^6.0
* SilverStripe Blog ^5.0

## Installation

```bash
composer require atwx/silverstripe-blog-extensions
```

## Configuration

Extensions can be configured in your project's YAML configuration files.

Example:
```yaml
SilverStripe\Blog\Model\BlogPost:
  extensions:
    - ATWX\BlogExtensions\Extensions\YourExtension
```
